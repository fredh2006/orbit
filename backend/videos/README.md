# Test Videos

This directory contains test video files for the LangGraph pipeline.

## Current Test Video

**`UGCVid1.MOV`** - Hardcoded default video for testing (9.4 MB)

This video is used by default in `test_pipeline.py`.

## Adding More Videos

Place any test videos here. Supported formats:
- .mp4
- .mov / .MOV
- .avi
- .mkv

**Note:** Video files are excluded from git via `.gitignore` to avoid bloating the repository.
